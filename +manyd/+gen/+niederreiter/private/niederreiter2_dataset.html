NIEDERREITER2_DATASET
Create a Niederreiter Dataset, Base = 2.

NIEDERREITER2_DATASET is a MATLAB program which creates a Niederreiter quasirandom dataset with base 2.

r = niederreiter2_dataset ( m, n, skip )
where
  • m is the spatial dimension;
  • n is the number of points to generate;
  • skip is the number of initial values to skip. A recommended value is 4096.
  • r is the computed m by n dataset.
The data is written to the file niederreiter2_m_n.txt.
